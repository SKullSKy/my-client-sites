Tactical Pro-Tips

Crosshair Placement

Always maintain reticle at head height relative to common holding angles. Minimizing vertical movement during engagement reduces reaction time.

Sound Propagation

Siege audio travels through the shortest opening. Use rotation holes and floor destruction to manipulate enemy information gathering.

Vertical Play

Control the floor above or below objectives to force defenders out of power positions. Never execute without clearing utility first.
